Как подключить Firebase

Без настройки Firebase авторизация не работает
  1. Зайдите на console.firebase.google.com и создайте проект
  2. В проекте: Build → Firestore Database → создайте базу (Start in test mode)
  3. Build → Authentication → Sign-in method → включите Email/Password
  4. В настройках проекта (⚙️ → Project settings) найдите раздел Your apps → Web app, скопируйте конфигурацию
  5. В этом файле найдите строку REPLACE_API_KEY и замените все REPLACE_* на свои значения
